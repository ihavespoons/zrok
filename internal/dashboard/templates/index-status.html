Setup

# Enable with a provider

# Build the index
zrok index build

# Natural language search
zrok semantic "authentication middleware"
# Explore related code
zrok semantic "SQL injection" --multi-hop

Maintenance

zrok index status # Check status
zrok index update # Update changed files
zrok index watch # Watch for changes
zrok index clear # Clear index
